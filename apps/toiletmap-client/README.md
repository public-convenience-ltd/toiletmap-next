# Toilet Map Client

The public-facing frontend for the Toilet Map, built with Astro and Preact.

## Stack

- **Framework**: [Astro](https://astro.build)
- **UI Library**: [Preact](https://preactjs.com)
- **Deployment**: Cloudflare Workers
- **Styling**: CSS Modules

## Development

```bash
pnpm install
pnpm run dev
```

## Deployment

```bash
pnpm run build
pnpm run deploy
```
